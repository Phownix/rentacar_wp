# Rent a Car [API]
_esta es la api de la app._



Applications require:
- node
- npm
- mongodb

NPM install: 
- express
- mongoose
- body-parser
- nodemon
- dotenv

Node Api: 
- [Crypto](https://nodejs.org/api/crypto.html)

vs-code extensions:
- REST Client



``api.http`` contiene las api-routes, usas la extension de vs-code "REST Client" para ver las peticiones
